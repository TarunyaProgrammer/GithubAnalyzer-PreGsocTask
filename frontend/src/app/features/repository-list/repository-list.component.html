Configure GitHub Webhook

To enable real-time updates for your repository, add a webhook in your GitHub Repository settings:

  1. Go to Settings > Webhooks > Add webhook.
  2. Payload URL: https://repo-arg-backend.onrender.com/api/webhooks/github
  3. Content type: application/json
  4. Secret: (Provided during setup)
  5. Events: Select Just the push event or Send me everything.
